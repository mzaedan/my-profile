# my-profile

Sebuah halaman profil sederhana yang berisi informasi dasar tentang pemilik proyek.

Deskripsi singkat:

- Proyek ini adalah template statis untuk halaman profil/portofolio kecil.
- Mudah disesuaikan dan siap dihosting di layanan hosting statis (GitHub Pages, Netlify, dsb.).

Fitur:

- Halaman profil utama: `index.html`
- Struktur sederhana, tanpa dependensi eksternal
- Siap untuk deploy statis

Cara menjalankan (lokal):

1. Buka folder proyek di PowerShell:

   ```powershell
   Set-Location 'd:\belajar-coding\paradaya\my-profile'
   ```

2. Untuk melihat langsung, buka `index.html` di browser (Windows):

   ```powershell
   Start-Process index.html
   ```

3. Atau jalankan web server sederhana untuk pengembangan:

   ```powershell
   python -m http.server 8000
   # lalu buka http://localhost:8000
   ```

Struktur proyek:

- `index.html` — Halaman profil utama
- `README.md` — Dokumentasi singkat (file ini)
